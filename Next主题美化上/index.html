前言

这里我用的 next 的版本是 next 5.1.4 同大家一同讨论此主题的美化。

且这个主题还有官网：

http://theme-next.iissnan.com/

有的更改需要从新生成一下静态的文件，最好在每一次下列更改的上传或开启本地测试之前执行下面的指令

打开终端

后面才执行

选择内置的不同的主题样式

  1. 打开文件

    打开 博客根目录/themes/next/_config.yml 文件

  2. 内置三种主题，通过注释一个，开启一个

更改头像

  1. 打开文件

    打开 博客根目录/themes/next/_config.yml 文件

  2. 找到这个配置

    然后取消一个控制头像的设置，通过相对路径可以放置新的图片，再改名即可，也可以使用网图

添加 ‘关于我’

  1. 在博客的根目录下打开终端，输入指令

  2. 会在指定文件夹下，生成一个 “about” 文件夹

    博客根目录/source/

    会在这个文件夹里面生成一个 “about” 文件夹，“about” 文件夹里面有一个 index.md 文件

  3. 打开 index.md 文件，在里面编写或更改内容即可

  4. 找到指定的配置文件

    打开 博客根目录/themes/next/_config.yml 文件

  5. 找到配置文件中的控制部分

    进行更改

    与文件夹名对应即可，增加相似功能，步骤一致

启用侧边栏的社交链接

    打开 博客根目录/themes/next/_config.yml 文件

  2. 找到配置的位置

    这里可以选择你需要用的（去掉注释，就能直接用），也可以直接添加

友情链接

  1. 打开指定的配置文件

    打开 博客根目录/themes/next/_config.yml 文件

  2. 找到指定的位置

    开启，更改成你的 URL ,同时可以继续添加友情链接
