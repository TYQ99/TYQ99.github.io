前言

这里是通过 git 、node.js 、hexo 的简单博客的安装、搭建、上传。但是没有更换默认的主题。

git 的安装

官网地址：

http://git-scm.com/

这个官网的地址的下载会非常慢建议在网上直接搜索 git下载会有一些 XXX软件园下载

当我们下载完成之后执行下列操作

  1. 下载完成后点击，安装程序，点击下一步
  1. 选择安装的路径，任何地方都行
  1. 下面基本上都是下一步即可
点击 Next点击 Next
  1. 最近的版本新加的选项，直接下一步就行
点 Next点 Next
  1. 继续下一步
可以根据提示选择点 Next
  1. 点击下一步
点 Next点 Next
  1. 点击下一步
点 Next点 Next
  1. 点击下一步
点 Next点 next
  1. 根据提示选择，点击安装，完成安装
两个对勾一般都不选，第一个完成安装后启动一下而已弹出的终端
  1. 第8步中两个都不选，也可以在桌面右键查看，但是有的电脑设置不允许添加到右键中，可以设置打开，也可以在安装目录，点击可执行的程序文件也行

当我们以上操作之后，打开终端，分别设置全局的git变量，用户名和邮箱

node.js 的安装

官方网址 ：

https://nodejs.org/zh-cn/download/

安装的 node.js 这个，包括两个部分，分别是 node.js 和 npm 包管理器

当我们下载完成之后执行下列操作

  1. 点击下载的可执行程序
点击 Next同意 并点击 Next
  1. 选择程序安装的路径，随意
路径随意，点击 Next点击 Next
  1. 下面默认下一步就行
点击 Next点击 安装
  1. 安装

检测 node.js 与 npm 是否正常安装

安装一个国内的镜像源**(可不用)**

由于是国外的速度会慢，所以我们可以用国内的镜像源，也可以不用，当我们安装以后的 npm 命令都可以换成 cnpm 命令来代替，效果都是一样的，但是速度会快一点。
